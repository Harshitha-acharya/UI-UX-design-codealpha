# 🍔 Food Delivery App UI/UX Design

## 📌 Project Overview

This project was created as part of the UI/UX Design assignment.

The objective was to design a Food Delivery Mobile Application using Figma by following the complete UI/UX design process, from low-fidelity wireframes to high-fidelity user interface designs.

**Design Tool:** Figma

---

# ✅ Task 1: Wireframing & Low-Fidelity Design

### Objective

Create low-fidelity wireframes for a mobile Food Delivery application.

### Screens Designed

- Splash Screen
- Login Screen
- Home Screen
- Restaurant Menu Screen
- Cart Screen
- Order Confirmation Screen

### Features Included

- Basic screen layouts
- Navigation flow
- Input fields
- Buttons
- Search bar
- Restaurant cards
- Cart layout

The wireframes focus only on structure and user flow without colors or images.

---

# ✅ Task 2: High-Fidelity UI Design

### Objective

Convert the wireframes into a professional user interface.

### Design Improvements

- Modern color palette
- Professional typography (Poppins)
- Rounded buttons
- Food images
- Icons using Iconify
- Proper spacing
- Restaurant cards
- Bottom navigation
- Attractive UI components

### Color Palette

| Color | Purpose |
|--------|----------|
| Orange (#FF6B35) | Primary |
| White (#FFFFFF) | Background |
| Green (#28A745) | Success |
| Light Gray (#F8F9FA) | Secondary Background |
| Dark Gray (#222222) | Text |
